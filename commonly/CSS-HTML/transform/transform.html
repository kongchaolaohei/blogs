# skew

在需要平行四边形的按钮时,如果有视图范围和交互范围一致时,skew就非常的好用了

<style>
  .parallelogram {
    height: 100px;
    width:  200px;
    position: relative;
    background-color: aqua;
    font-size: 30px;
    transform: skewX(-45deg);
    transform-origin: bottom left;
  }
  .parallelogram>span {
    position: absolute;
    top: 50%;
    left: calc(50% + 21px);
    transform: skewX(45deg) translate(-50%, -50%);
  }
</style>
<div class="parallelogram">
  <span>parallelogram</span>
</div>
